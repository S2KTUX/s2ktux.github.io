Descripción

Domina los comandos esenciales de Linux para principiantes. Este curso te enseñará a usar la terminal de forma eficiente.

Comandos Básicos

En esta sección aprenderás:

  • Uso de comandos como ls, cd, y pwd.
  • Explorar el sistema de archivos.

Gestión de Archivos

Aprenderás a:

  • Crear, mover y eliminar archivos y directorios.
  • Comandos para monitoreo del sistema (du, df).

Duración

3 semanas

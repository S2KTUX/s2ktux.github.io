106.1 Instalar y configurar X11

El Sistema X Window (X11) es la base para interfaces gráficas en Linux, proporcionando una arquitectura cliente-servidor para mostrar texto y gráficos.

Arquitectura de X11

X11 utiliza una arquitectura cliente-servidor:

  • Clientes X: Aplicaciones (terminales, juegos) que solicitan al servidor dibujar ventanas.
  • Servidor X: Gestiona la pantalla, colocando dibujos y manejando entradas (ratón, teclado).

X11 permite configuraciones en red. Cada servidor X tiene un nombre de pantalla (hostname:displaynumber.screennumber), almacenado en la variable DISPLAY (ej. :0 local, localhost:10.0 remoto). Gestores de pantalla como GDM, SDDM o LightDM inician sesiones X y autentican usuarios.

Extensiones como libXrandr, libXcursor o libX11 añaden funcionalidades.

Configuración del servidor X

El archivo principal es /etc/X11/xorg.conf, aunque en distros modernas se autoconfigura. El archivo se divide en secciones Section "Nombre" ... EndSection.

  • Archivos adicionales en /etc/X11/xorg.conf.d/ (usuario) y /usr/share/X11/xorg.conf.d/ (distribución).
  • Secciones comunes: InputDevice/InputClass (teclados/ratones), Monitor (monitores), Device (tarjeta gráfica), Screen (vínculo monitor-dispositivo), ServerLayout (agrupación).
  • Generar configuración: sudo Xorg -configure (crea xorg.conf.new).
  • Teclado temporal: setxkbmap -model chromebook -layout "gr(polytonic)".
  • Teclado permanente: localectl set-x11-keymap "gr(polytonic)" chromebook.

Herramientas y archivos clave

  • ~/.xsession-errors: Log de errores de sesiones X.
  • xhost: Controla acceso (ej. xhost +local: permite locales).
  • xauth: Gestiona autenticación con cookies en ~/.Xauthority.
  • xdpyinfo: Muestra info del servidor (pantalla, versión, extensiones).

Wayland

Protocolo moderno que reemplaza X11. Más ligero; los clientes renderizan directamente y un compositor gestiona E/S. Usa variable WAYLAND_DISPLAY. Aplicaciones X11 pueden correr en XWayland.

106.2 Escritorios gráficos

Los escritorios gráficos en Linux se basan en X11 y ofrecen interfaces visuales con entornos como Gnome, KDE y Xfce.

Sistema X Window

  • Maneja eventos de entrada (ratón, teclado).
  • Copia y pega entre aplicaciones.
  • Interfaz para dibujar elementos gráficos.
  • No crea efectos complejos (lo hacen las apps).

La variable DISPLAY indica qué servidor X usar. El archivo xorg.conf organiza ajustes en secciones (Monitor, Device, Screen).

Ambientes de escritorio

Incluyen:

  • Gestor de ventanas: Controla posición, barras y botones.
  • Widgets: Elementos (botones, campos) usando GTK+ o Qt.
  • Apps básicas: Terminal, gestor archivos, navegador.

Ofrecen coherencia visual.

Entornos populares

  • Gnome: Fedora, Ubuntu. Usa Gnome Shell (pantalla completa) o Classic.
  • KDE: openSUSE, Kubuntu. Usa KDE Plasma (basado en Qt).
  • Xfce: Ligero y modular, ideal para pocos recursos.

Otros: Cinnamon, MATE, LXDE.

Acceso remoto

  • XDMCP: Protocolo pantallas remotas (poco usado, alto ancho banda/inseguro).
  • VNC: Usa RFB, puertos 5900+, sin cifrado nativo (requiere SSH/VPN).
  • Spice: Para virtualizados. Integra dispositivos locales (USB) y proporciona acceso gráfico fluido.
  • RDP: Acceso a Windows (soportado en Linux con clientes abiertos).

Herramienta: Remmina (soporta XDMCP, VNC, RDP, Spice).

106.3 Accesibilidad

La accesibilidad en Linux permite ajustar el sistema para usuarios con necesidades especiales.

Configuración de accesibilidad

  • Gnome: Acceso universal (menú esquina superior).
  • KDE: Configuración del sistema → Accesibilidad.
  • Xfce: Ajustes gráficos → Accesibilidad (menos funciones).

Gestores de pantalla (XDM, KDM, GDM) inician el entorno y gestionan el login, permitiendo activar funciones como lectores de pantalla o alto contraste.

Asistencia de teclado y mouse

  • Teclas pegajosas: Atajos uno a uno (ej. Ctrl, luego C).
  • Teclas lentas: Mantener tecla para aceptar.
  • Teclas de rebote: Retardo para evitar pulsaciones accidentales.
  • Teclas de conmutación: (KDE) pulsar 2 veces Alt/Ctrl/Shift (como Bloq Mayús).
  • Teclas de repetición: Ajusta repetición al mantener pulsada.
  • Teclas de ratón: Controla puntero con teclado numérico (5 = clic izq).
  • Gestos: Shift 5 veces (pegajosas), Shift 8 seg (lentas).
  • Teclado en pantalla: Gnome (nativo), KDE (paquete onboard).

Estas opciones están en AccessX (extensión X11), ajustables con xkbset.

Deficiencias visuales

  • Temas de alto contraste/impresión grande: Colores nítidos y texto grande.
  • Lupa de pantalla: Zoom (Gnome: Zoom; KDE: KMagnifier; Xfce: Alt + rueda).
  • Lector de pantalla (Orca): Lee texto en voz o Braille, permitiendo interacción a usuarios ciegos o con visión reducida.
  • Pantalla braille: Orca soporta dispositivos que representan texto con puntos táctiles.

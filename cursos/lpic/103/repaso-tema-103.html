103.1 Trabajar desde la línea de comandos

  • Información del sistema:
    • pwd: Muestra el directorio actual (ej. /home/usuario).
    • touch archivo.txt: Crea un archivo vacío.
    • ls: Lista archivos (ej. archivo.txt documentos).
    • uname -a: Información del sistema (ej. kernel, arquitectura).
  • Información de comandos:
    • man ls: Manual del comando.
    • apropos copy: Busca comandos relacionados (ej. cp).
    • type cd: Tipo de comando (ej. shell builtin).
    • which ls: Ruta del comando (ej. /bin/ls).
    • history: Lista comandos ejecutados.
  • Variables de entorno:
    • env: Muestra variables (ej. PATH, HOME).
    • echo $HOME: Valor de variable (ej. /home/usuario).
    • export MI_VAR="hola": Crea variable.
    • unset MI_VAR: Elimina variable.
  • Caracteres especiales: Usa comillas o \ para espacios. Ejemplo: touch 'my file'.

103.2 Procesar secuencias de texto usando filtros

  • Redirecciones y tuberías:
    • echo "Hola" > file.txt: Sobrescribe.
    • echo "Adiós" >> file.txt: Añade.
    • ls | grep ".txt": Filtra archivos.
  • Archivos comprimidos:
    • gzip file.txt: Comprime (crea file.txt.gz).
    • zcat file.txt.gz: Muestra contenido comprimido.
  • Ver archivos:
    • less file.txt: Navegación interactiva.
    • head -n 3 file.txt: Primeras 3 líneas.
    • tail -n 2 file.txt: Últimas 2 líneas.
  • sed:
    • sed 's/cat/dog/' file.txt: Reemplaza.
    • sed -i 's/cat/dog/' file.txt: Reemplaza directamente.
  • Otros:
    • wc -l file.txt: Cuenta líneas.
    • sort -n file.txt: Ordena numéricamente.
    • sha256sum file.txt: Genera hash para verificar integridad.

103.3 Administración básica de archivos

  • Listar:
    • ls -la: Muestra archivos ocultos y detalles.
    • ls -R: Lista recursiva.
  • Manipulación:
    • touch file.txt: Crea archivo.
    • cp file.txt copia.txt: Copia.
    • mv file.txt /tmp/: Mueve.
    • rm -r dir: Elimina.
    • mkdir -p dir/subdir: Crea directorios.
  • Comodines:
    • *: Cero o más (ej. ls *.txt).
    • ?: Un carácter (ej. ls file?.txt).
  • find:
    • find / -name "file.txt": Por nombre.
    • find /home -size +5M: Por tamaño.
    • find /tmp -mtime +2: Por tiempo.
  • Archivado:
    • tar -cvf file.tar file1.txt: Crea archivo.
    • tar -czvf file.tar.gz dir: Comprime con gzip.

103.4 Uso de secuencias, tuberías y redirecciones

  • Redirecciones:
    • wc -l < file.txt: Entrada.
    • ls > lista.txt: Salida (sobrescribe).
    • ls 2> errores.txt: Errores.
    • ls &> salida.txt: Salida y errores.
  • Here Document/String:
    • wc -c << EOF: Entrada multilínea.
    • wc -c <<< "texto": Entrada de una línea.
  • Tuberías:
    • ls | grep ".txt": Filtra.
    • ps aux | grep "python" | sort -k 3 -r: Filtra y ordena.
  • Sustitución:
    • echo $(ls): Usa salida de comando.
    • find . -name "*.log" | xargs rm: Pasa resultados a otro comando.

103.5 Crear, supervisar y matar procesos

  • Control de trabajos:
    • jobs: Lista trabajos.
    • fg %1: Trae a primer plano.
    • sleep 500 &: Ejecuta en segundo plano.
    • kill -9 1234: Termina proceso.
  • Monitoreo:
    • ps -aux: Lista todos los procesos.
    • top: Vista en tiempo real.
    • pgrep python: Busca procesos.
  • Multiplexores:
    • screen -S sesion: Crea una sesión de screen (usa Ctrl+a d para desconectar).
    • tmux new -s sesion: Crea una sesión de tmux (usa Ctrl+b d para desconectar).

103.6 Modificar la prioridad de ejecución de los procesos

  • Prioridad: Valor nice de -20 (alta) a 19 (baja). Por defecto: 0.
  • Comandos:
    • nice -n 10 comando: Inicia con nice 10.
    • renice 15 1234: Cambia nice del PID 1234.
  • Ver prioridades: top, ps -el.

103.7 Realizar búsquedas en archivos de texto usando expresiones regulares

  • Expresiones básicas:
    • .: Cualquier carácter.
    • ^: Inicio de línea.
    • $: Final de línea.
  • Corchetes: [a-z], [[:digit:]].
  • Cuantificadores: * (0 o más), + (1 o más, con egrep).
  • grep:
    • grep -i "hola" file.txt: Ignora mayúsculas.
    • egrep "Hola|Adios" file.txt: Ramas.
  • find: find / -regex ".*/\..*".
  • sed: sed 's/gato/perro/g' file.txt.

103.8 Edición básica de archivos

  • vi:
    • vi archivo.txt: Abre un archivo en vi.
    • Modos:
      • Normal: Navegación y comandos.
      • Inserción: Escribir texto (i para entrar, Esc para salir).
  • Comandos vi:
    • h j k l: Moverse.
    • yy, p: Copiar y pegar.
    • :w, :q: Guardar y salir.
  • Editores alternativos:
    • nano archivo.txt: Editor simple (Ctrl+O guardar, Ctrl+X salir).
    • emacs archivo.txt: Editor avanzado (Ctrl+X Ctrl+S guardar, Ctrl+X Ctrl+C salir).

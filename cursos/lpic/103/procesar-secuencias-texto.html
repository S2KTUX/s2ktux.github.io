Redirecciones y tuberías (pipes)

Los comandos de texto usan stdin (entrada), stdout (salida) y stderr (errores). Podemos redirigir estas salidas.

  • >: Redirige la salida a un archivo (sobrescribe).
    Ejemplo: echo "Hola" > saludo.txt crea saludo.txt con "Hola".
  • >>: Añade la salida a un archivo (sin sobrescribir).
    Ejemplo: echo "Adiós" >> saludo.txt añade "Adiós" al final de saludo.txt.
  • | (pipe): Envía la salida de un comando a otro.
    Ejemplo: ls | grep ".txt" filtra archivos con ".txt".
  • diff: Compara dos archivos.
    Ejemplo: diff archivo1.txt archivo2.txt muestra las diferencias.
  • grep: Filtra texto. Con -i ignora mayúsculas/minúsculas.
    Ejemplo: grep -i "hola" saludo.txt busca "hola" sin importar mayúsculas.

Leer un archivo comprimido

Puedes comprimir, descomprimir y ver archivos comprimidos con estos comandos.

  • bzip2 / bunzip2: Comprime y descomprime.
    Ejemplo: bzip2 archivo.txt crea archivo.txt.bz2. Usa bunzip2 archivo.txt.bz2 para descomprimir.
  • gzip / gunzip: Comprime y descomprime.
    Ejemplo: gzip archivo.txt crea archivo.txt.gz. Usa gunzip archivo.txt.gz para descomprimir.
  • xz / unxz: Comprime y descomprime.
    Ejemplo: xz archivo.txt crea archivo.txt.xz. Usa unxz archivo.txt.xz para descomprimir.
  • bzcat, zcat, xzcat: Muestra el contenido sin descomprimir.
    Ejemplo: zcat archivo.txt.gz muestra el contenido de archivo.txt.gz.

Ver un archivo paginado

Estos comandos te permiten leer archivos página por página.

  • less: Muestra el archivo con navegación (arriba/abajo).
    Ejemplo: less archivo.txt te deja navegar por archivo.txt.
  • more: Muestra el archivo página por página (solo hacia adelante).
    Ejemplo: more archivo.txt muestra archivo.txt página por página.

Obtener una parte del texto

Estos comandos extraen partes específicas de un archivo.

  • head: Muestra las primeras líneas. Usa -n para elegir cuántas.
    Ejemplo: head -n 3 archivo.txt muestra las primeras 3 líneas.
  • tail: Muestra las últimas líneas. Usa -n para elegir cuántas.
    Ejemplo: tail -n 2 archivo.txt muestra las últimas 2 líneas.

Fundamentos de sed

sed filtra y transforma texto.

  • sed (borrar): Elimina líneas que coincidan.
    Ejemplo: sed '/cat/d' nombres.txt elimina líneas con "cat".
  • sed (imprimir coincidencias): Con -n y /patrón/p, imprime solo las líneas que coinciden.
    Ejemplo: sed -n '/cat/p' ftu.txt muestra solo las líneas de ftu.txt que contienen "cat".
  • sed (reemplazar): Sustituye texto.
    Ejemplo: sed 's/cat/dog/' nombres.txt cambia "cat" por "dog".
  • sed (reemplazar todas las apariciones): Sustituye todas las apariciones en el archivo.
    Ejemplo: sed -i 's/cat/dog/g' nombres.txt reemplaza todas las apariciones de "cat" por "dog" en nombres.txt.
  • sed (con backup): Reemplaza y crea un respaldo.
    Ejemplo: sed -i.backup 's/cat/dog/' nombres.txt cambia y crea nombres.txt.backup.

Otros comandos útiles

Estos comandos también son útiles para manejar texto.

  • nl: Numera las líneas del archivo.
    Ejemplo: nl archivo.txt muestra 1 Hola 2 Adiós.
  • wc: Cuenta líneas, palabras o caracteres. Con -l solo cuenta líneas.
    Ejemplo: wc -l archivo.txt podría mostrar 2 archivo.txt (2 líneas).
  • cut: Extrae partes de cada línea.
    Ejemplo: cut -d' ' -f1 nombres.txt extrae la primera palabra de cada línea (separador espacio).
  • sort: Ordena líneas. Con -n ordena numéricamente.
    Ejemplo: sort -n numeros.txt ordena 2 5 10 en orden numérico.
  • tr: Sustituye caracteres.
    Ejemplo: echo "hola" | tr 'a' 'A' muestra holA.

Garantizar la integridad de los datos

Verifica que los archivos no hayan sido alterados.

  • sha256sum: Genera o verifica un hash.
    Ejemplo:
    • sha256sum archivo.txt genera un hash.
    • sha256sum archivo.txt > hash.txt guarda el hash.
    • sha256sum -c hash.txt verifica si archivo.txt coincide.

Buscar más en los archivos

Explora archivos en diferentes formatos (octal, ASCII, etc.).

  • od: Muestra el contenido en formatos como octal o ASCII.
    Ejemplo: od -c archivo.txt muestra los caracteres en ASCII: 0000000 H o l a \n.

Alternativas: xxd o hexdump.

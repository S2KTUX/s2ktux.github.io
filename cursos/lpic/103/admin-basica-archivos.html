Manipulación de archivos

En Linux, todo es un archivo: archivos regulares (datos o programas), directorios y archivos especiales (como dispositivos).

Usar ls para listar archivos

ls muestra información sobre archivos y directorios.

  • ls -l: Muestra permisos, propietario, tamaño, fecha y nombre.
    Ejemplo: ls -l podría mostrar: -rw-r--r-- 1 usuario grupo 1024 Apr 26 12:00 archivo.txt.
    - Primer carácter: - (archivo), d (directorio), l (enlace simbólico).
  • Opciones útiles:
    • -a: Muestra archivos ocultos (que empiezan con .).
      Ejemplo: ls -a muestra .bashrc.
    • -h: Tamaños en formato legible (KB, MB).
      Ejemplo: ls -lh muestra 1.0K en lugar de 1024.
    • -R: Lista directorios de forma recursiva.
      Ejemplo: ls -R muestra el contenido de subdirectorios.

Crear, copiar, mover y eliminar

  • touch: Crea un archivo vacío.
    Ejemplo: touch nuevo.txt crea nuevo.txt.
  • cp: Copia archivos. Usa -r para directorios.
    Ejemplo: cp archivo.txt copia.txt o cp -r dir1 dir2.
  • mv: Mueve o renombra archivos.
    Ejemplo: mv archivo.txt /tmp/ mueve archivo.txt a /tmp.
  • rm: Elimina archivos. Usa -r para directorios.
    Ejemplo: rm archivo.txt o rm -r dir1.
  • mkdir: Crea directorios. Usa -p para crear subdirectorios.
    Ejemplo: mkdir -p dir1/subdir crea subdir dentro de dir1.
  • rmdir: Elimina directorios vacíos.
    Ejemplo: rmdir dir1 elimina dir1 si está vacío.

Archivos globbing y wildcards

Los comodines permiten coincidir con varios archivos usando caracteres especiales.

  • *: Coincide con cero o más caracteres.
    Ejemplo: ls *.txt lista todos los archivos que terminan en .txt.
  • ?: Coincide con un solo carácter.
    Ejemplo: ls file?.txt coincide con file1.txt o file2.txt.
  • []: Coincide con un carácter dentro de un rango o lista.
    Ejemplo: ls file[1-3].txt coincide con file1.txt, file2.txt, file3.txt.

Cómo encontrar archivos

Usa find para buscar archivos según diferentes criterios.

  • Por nombre: Busca por nombre. Usa -iname para ignorar mayúsculas.
    Ejemplo: find / -name "archivo.txt" busca archivo.txt.
  • Por tipo: Busca según el tipo (f para archivo, d para directorio, l para enlace).
    Ejemplo: find / -type d busca directorios.
  • Por tamaño: Busca por tamaño (K, M, G). Usa + o - para mayor/menor.
    Ejemplo: find /home -size +5M busca archivos mayores a 5 MB.
  • Por tiempo de modificación: Busca por última modificación (-mtime en días, -mmin en minutos).
    Ejemplo: find /tmp -mtime +2 busca archivos modificados hace más de 2 días.
  • Ejecutar acción: Usa -exec para ejecutar un comando con los resultados.
    Ejemplo: find /var/log -type f -name "*.log" -mtime +2 -exec ls -l {} \; lista archivos .log modificados hace más de 2 días.

Archivado de archivos

Usa tar para archivar archivos (no los comprime, solo los agrupa).

  • Crear archivo: -c (crear), -v (detalles), -f (archivo).
    Ejemplo: tar -cvf archivo.tar archivo1.txt archivo2.txt crea archivo.tar.
  • Extraer archivo: -x (extraer).
    Ejemplo: tar -xvf archivo.tar extrae los archivos.
  • Listar contenido: -t (listar).
    Ejemplo: tar -tvf archivo.tar muestra el contenido de archivo.tar.
  • Comprimir con tar:
    • -z: Comprime con gzip.
      Ejemplo: tar -czvf comprimido.tar.gz dir1 crea comprimido.tar.gz.
    • -j: Comprime con bzip2.
      Ejemplo: tar -cjvf comprimido.tar.bz2 dir1 crea comprimido.tar.bz2.

Otros comandos:

  • cpio: Archiva y extrae.
    Ejemplo: ls | cpio -o > archivo.cpio crea un archivo; cpio -id < archivo.cpio lo extrae.
  • dd: Copia datos a bajo nivel.
    Ejemplo: dd if=/dev/zero of=/tmp/archivo bs=100 count=10 crea un archivo de 1000 bytes.

107.1 Administración de usuarios y grupos

Gestiona cuentas de usuario/grupo y archivos relacionados.

  • Comandos:
    • useradd: Crea cuenta (ej. -m crea home, -s define shell).
    • usermod: Modifica cuenta (ej. -G grupos, -L bloquea).
    • userdel: Elimina usuario (-r borra home).
    • groupadd: Crea grupo (-g para GID).
    • groupmod, groupdel: Modifica/elimina grupos.
    • passwd: Cambia contraseñas (root para otros, usuario regular para la propia).
    • chage: Gestiona caducidad de contraseñas.
    • getent: Consulta bases de datos (ej. getent passwd sysriser).
  • Archivos:
    • /etc/passwd: Info usuarios (7 campos: nombre, passwd x, UID, GID, desc, home, shell).
    • /etc/group: Info grupos (4 campos: nombre, passwd x, GID, miembros).
    • /etc/shadow: Contraseñas cifradas (legible solo root).
    • /etc/gshadow: Contraseñas cifradas de grupos.
    • /etc/skel: Plantilla para homes de nuevos usuarios.
    • /etc/login.defs: Configuración (ej. CREATE_HOME).

107.2 Automatización de tareas

Programa tareas con cron, systemd, at y systemd-run.

  • Cron: Ejecuta tareas periódicas.
    • Crontab: 6 campos (min, hora, día mes, mes, día sem, comando). Archivos usuario en /var/spool/cron/.
    • Usuario: Editar crontab -e, ver crontab -l.
    • Sistema: /etc/crontab y /etc/cron.d/ (con campo usuario).
    • Restricciones: /etc/cron.allow (permite listados), /etc/cron.deny (prohíbe listados).
  • Systemd timers: Alternativa a cron, usa .timer con OnCalendar.
  • at: Tareas de ejecución única. Usa atq (listar) y atrm (eliminar).
  • systemd-run: Tareas únicas con --on-calendar o --on-active.

107.3 Localización e internacionalización

Configura zonas horarias y localización.

  • Zonas horarias:
    • /etc/timezone, /etc/localtime: Definen zona (/etc/localtime enlace a /usr/share/zoneinfo/).
    • TZ: Variable para zona de sesión.
    • Comandos: date (muestra), timedatectl (configura), tzselect (selecciona).
  • Localización:
    • LANG: Idioma/codificación (ej. es_ES.UTF-8).
    • LC_*: Formatos específicos (ej. LC_TIME, LC_MONETARY).
    • LC_ALL: Anula todas las variables de localización.
    • /etc/locale.conf: Configuración del sistema.
    • Comandos: locale (muestra), localectl (configura).
  • Conversión de codificación:
    • iconv: Convierte codificaciones. Listar con iconv -l.

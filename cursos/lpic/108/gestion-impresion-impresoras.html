108.4 Gestión de la impresión y las impresoras

Descubre cómo gestionar impresoras en Linux con CUPS y herramientas legacy de forma sencilla.

A pesar de la idea de una sociedad sin papel, la impresión sigue siendo clave en muchas organizaciones. En Linux, el sistema CUPS (Common Unix Printing System) facilita imprimir y gestionar impresoras, enviando archivos a una cola, procesándolos con filtros y enviándolos a la impresora.

Servicio CUPS y su configuración

CUPS es un sistema de impresión común en Linux, y se puede instalar fácilmente en distribuciones como Debian con `sudo apt install cups` o en Fedora con `sudo dnf install cups` seguido de `sudo systemctl start cups.service`. Para verificar que funciona, usa `systemctl status cups.service`.

La configuración principal se encuentra en el directorio `/etc/cups/`, que centraliza los archivos de configuración del sistema de impresión, incluyendo archivos clave como:

  • /etc/cups/cupsd.conf: Configura el servicio, como el acceso a colas o si la interfaz web está activa (ejemplo: `WebInterface Yes`).
  • /etc/cups/printers.conf: Almacena las configuraciones de las impresoras gestionadas por CUPS, con secciones `` para cada una (no edites mientras CUPS está activo).
  • /etc/printcap: Archivo legacy de LPD, creado por CUPS como enlace simbólico a `/run/cups/printcap`, lista las impresoras disponibles.
  • /etc/cups/ppd/: Directorio con archivos PPD que describen las capacidades de las impresoras.

Los logs, que registran accesos, trabajos y errores, se guardan en `/var/log/cups/`, útiles para diagnosticar problemas.

La interfaz web (http://localhost:631 si está habilitada) permite gestionar impresoras y colas, requiriendo autenticación de root para cambios. Ejemplo de configuración de acceso: ` AuthType Default Require user @SYSTEM `.

Herramientas legacy de lpd

Las herramientas legacy de lpd, compatibles con CUPS, permiten gestionar impresiones de forma básica:

  • lpr: Envía archivos a la impresora predeterminada. Ejemplo: `lpr documento.txt` imprime en la impresora por defecto.
  • lprm: Cancela trabajos de impresión en curso, eliminando tareas específicas de la cola. Ejemplo: `lprm 20` elimina el trabajo con ID 20.
  • lpq: Muestra una lista de trabajos en la cola de impresión, incluyendo detalles como IDs y estados. Ejemplo: `lpq -a` lista colas de todas las impresoras.
  • lpstat: Ofrece detalles sobre el estado de las impresoras y trabajos en cola. Ejemplo: `lpstat -p` muestra impresoras disponibles.

Estas herramientas son útiles en sistemas sin entorno gráfico para enviar, consultar o cancelar trabajos de impresión.

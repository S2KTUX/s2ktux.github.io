108.1 Configuración del tiempo

Configura y sincroniza la hora en el sistema.

  • Comandos:
    • timedatectl: Muestra/ajusta hora, fecha, zona. Ej: timedatectl set-timezone Europe/Madrid.
    • hwclock: Gestiona reloj hardware (RTC). Ej: hwclock --systohc.
    • date: Muestra/ajusta hora actual.
    • chronyd o ntpd: Sincroniza NTP.
  • Archivos:
    • /etc/timezone: Define zona horaria.
    • /etc/localtime: Enlace a /usr/share/zoneinfo/.
    • /etc/chrony.conf / /etc/ntp.conf: Servidores NTP.

108.2 Gestión del sistema

Administra servicios, logs y recursos del sistema.

  • Comandos:
    • systemctl: Controla servicios (start/enable).
    • journalctl: Consulta journal (filtros por servicio/tiempo).
    • logger: Añade mensajes al log (ej. logger "Mensaje de prueba").
    • df -h: Uso de disco.
    • du -sh /ruta: Tamaño de directorios.
    • top / htop: Monitoriza procesos.
    • kill PID / pkill nombre: Termina procesos.
    • dmesg: Mensajes kernel.
  • Archivos:
    • /etc/rsyslog.conf: Configuración de rsyslog.
    • /var/log/: Directorio de logs (syslog, auth.log).

108.3 Agente de transferencia de correo

Gestiona envío y recepción de correos con MTAs.

  • Comandos:
    • sendmail: Comando compatible MTAs. Ej: sendmail -q reintenta envíos.
    • mailq: Muestra cola pendiente (ej. "Connection refused").
    • newaliases: Actualiza alias tras editar /etc/aliases.
  • Archivos:
    • /etc/aliases: Alias de correo a nivel de sistema.
    • ~/.forward: Redirige correos del usuario.
    • /var/spool/mail/: Buzones de usuarios.
  • Conceptos: MTAs (sendmail, Postfix, Exim), SMTP (puerto 25), DNS MX.

108.4 Gestión de la impresión

Configura e imprime con CUPS y herramientas legacy.

  • Comandos:
    • apt install cups / dnf install cups: Instala CUPS.
    • systemctl start cups.service: Inicia servicio.
    • lpr archivo: Imprime archivo.
    • lpq -a: Muestra colas y trabajos.
    • lpstat -p: Verifica impresoras.
    • lprm ID: Cancela trabajo (ej. lprm 20).
  • Archivos:
    • /etc/cups/: Configuración CUPS (cupsd.conf, printers.conf).
  • Interfaz: http://localhost:631.

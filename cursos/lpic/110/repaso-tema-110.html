110.1 Tareas de administración de seguridad

Auditoría de permisos y gestión de cuentas.

  • Permisos Especiales:
    - SUID (4000): Ejecuta con permisos del propietario. Ej: passwd.
    - SGID (2000): Ejecuta con permisos del grupo. En directorios, hereda grupo.
    - Buscar: find / -perm -4000 (SUID) o -perm -2000 (SGID).
  • Contraseñas:
    - Hashes en /etc/shadow. /etc/passwd tiene una x.
    - chage -l usuario (listar info), chage -M 30 usuario (max días).
    - /etc/nologin impide logins no root.
  • Recursos y Puertos:
    - ulimit -a (ver límites). Configurar en /etc/security/limits.conf.
    - lsof -i :80, netstat -tulpn o ss -tulpn para puertos.
  • Sudo: Editar con visudo. Sintaxis: usuario HOST=(USUARIOS) COMANDOS.

110.2 Configuración de la seguridad del sistema

Hardening de servicios de red.

  • Superdemonios:
    - xinetd: Config en /etc/xinetd.d/.
    - systemd.socket: Escucha y levanta el servicio bajo demanda. Ej: systemctl start ssh.socket.
  • Servicios: Deshabilitar innecesarios. systemctl disable --now servicio.
  • TCP Wrappers:
    - Archivos: /etc/hosts.allow (permite) y /etc/hosts.deny (deniega).
    - Regla: Allow tiene prioridad sobre Deny.
    - Verificar soporte: ldd /usr/sbin/sshd | grep libwrap.

110.3 Protección de datos mediante cifrado

Seguridad en redes y archivos.

  • SSH:
    - Generar claves: ssh-keygen -t ecdsa -b 521.
    - Clave pública a ~/.ssh/authorized_keys en el servidor.
    - ssh-agent y ssh-add para gestión de passphrase.
  • Túneles SSH:
    - Local: ssh -L puerto_local:host_remoto:puerto_remoto user@host.
    - Remoto: ssh -R puerto_remoto:host_local:puerto_local user@host.
    - X11: ssh -X user@host.
  • GPG:
    - Generar: gpg --gen-key.
    - Cifrar: gpg --encrypt --recipient user archivo.
    - Descifrar: gpg --decrypt archivo.gpg.
    - Firmar: gpg --sign archivo.
    - Verificar: gpg --verify archivo.sig.

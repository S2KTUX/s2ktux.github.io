Gestión con RPM, YUM, DNF y Zypper

En sistemas Red Hat, Fedora y SUSE se utilizan diferentes herramientas para gestionar paquetes. RPM es de bajo nivel y trabaja con archivos .rpm, pero no resuelve dependencias. YUM y DNF son de alto nivel, perfectos para Red Hat y Fedora, mientras que Zypper es la opción en SUSE. Aquí tienes los comandos principales para cada una.

Comandos con RPM

Acción Comando
Instalar un paquete rpm -i paquete.rpm (o rpm -ivh)
Actualizar un paquete rpm -U paquete.rpm (instala/actualiza)
rpm -F paquete.rpm (solo actualiza)
Eliminar un paquete rpm -e paquete
Listar paquetes instalados rpm -qa
Ver información de un paquete rpm -qi paquete (instalado)
rpm -qip paquete.rpm (no instalado)
Listar archivos de un paquete rpm -ql paquete (instalado)
rpm -qlp paquete.rpm (no instalado)
Buscar a qué paquete pertenece un archivo rpm -qf /ruta/archivo

Comandos con YUM

Acción Comando
Instalar un paquete yum install paquete
Actualizar un paquete yum update paquete
Eliminar un paquete yum remove paquete
Ver información de un paquete yum info paquete
Buscar a qué paquete pertenece un archivo yum whatprovides archivo
Buscar un paquete yum search paquete
Actualizar el índice de paquetes yum check-update
Limpiar caché de paquetes yum clean all (en /var/cache/yum)

Comandos con DNF

Acción Comando
Instalar un paquete dnf install paquete
Actualizar un paquete dnf upgrade paquete
Eliminar un paquete dnf remove paquete
Listar paquetes instalados dnf list --installed
Ver información de un paquete dnf info paquete
Listar archivos de un paquete dnf repoquery -l paquete
Buscar a qué paquete pertenece un archivo dnf provides archivo
Buscar un paquete dnf search paquete

Comandos con Zypper

Acción Comando
Instalar un paquete zypper in paquete
Actualizar un paquete zypper update paquete
Eliminar un paquete zypper rm paquete
Ver información de un paquete zypper info paquete
Buscar a qué paquete pertenece un archivo zypper se --provides archivo
Buscar un paquete zypper se paquete
Actualizar el índice de paquetes zypper ref
Listar actualizaciones disponibles zypper list-updates

Configuración de repositorios

Cuando trabajo con repositorios, suelo gestionarlos en /etc/yum.repos.d/ para YUM, DNF y Zypper. Aquí están los comandos que utilizo para cada herramienta:

  • YUM: Listo todos los repositorios con yum repolist all, añado uno nuevo con yum-config-manager --add-repo URL.repo, y habilito o deshabilito con yum-config-manager --enable/--disable.
  • DNF: Muestro los repositorios con dnf repolist, añado uno con dnf config-manager --add-repo URL.repo, y habilito o deshabilito con dnf config-manager --set-enabled/--set-disabled ID_REPO.
  • Zypper: Listo los repositorios con zypper repos, añado uno con zypper addrepo URL ID_REPO, habilito o deshabilito con zypper modifyrepo -e/-d ID_REPO, activo o desactivo autorefresh con zypper modifyrepo -F/-f ID_REPO, y elimino con zypper removerepo ID_REPO.

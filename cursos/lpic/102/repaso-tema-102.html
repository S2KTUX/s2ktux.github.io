102.1 Particiones y sistemas de archivos

Saber cómo gestionar discos y particiones:

  • Crear particiones con fdisk o parted, y sistemas de archivos con mkfs.
  • Montar y desmontar con mount y umount. Configurar montajes automáticos en /etc/fstab.
  • Gestionar espacio con df (uso del disco) y du (tamaño de archivos).
  • Crear y activar swap con mkswap y swapon.

102.2 Instalación de un gestor de arranque

GRUB para arrancar Linux:

  • GRUB 2: Instalamos con grub-install. Configuramos en /etc/default/grub y generamos /boot/grub/grub.cfg con grub-mkconfig. Incluye archivos como /boot/grub/i386-pc/lvm.mod (para soporte LVM) y /boot/grub/grub.cfg (configuración principal).
  • GRUB Legacy: Configuramos en /boot/grub/menu.lst. Discos/particiones se numeran desde 0 (ej. hd0,0).
  • La partición /boot contiene el kernel, disco RAM (initrd.img), y archivos de GRUB.

102.3 Bibliotecas compartidas

Las bibliotecas compartidas (.so) permiten a los programas compartir código:

  • Están en /lib o /usr/lib. Usamos ldd para ver dependencias de un programa.
  • Configuramos rutas en /etc/ld.so.conf y actualizamos con ldconfig.
  • Podemos añadir rutas temporales con la variable LD_LIBRARY_PATH.

102.4 Gestión de paquetes Debian

Gestion de paquetes en Debian con estas herramientas:

  • dpkg (bajo nivel): dpkg -i paquete.deb (instalar), dpkg -r/--purge paquete (eliminar/purgar), dpkg -l (listar).
  • apt (alto nivel): apt install paquete, apt update, apt upgrade, apt remove/purge paquete. Alternativamente, apt-get remove paquete elimina un paquete dejando archivos de configuración.
  • Repositorios en /etc/apt/sources.list y /etc/apt/sources.list.d/. Actualizamos con apt update.

102.5 Gestión de paquetes RPM y YUM

El uso de varias herramientas de gestion de paquetes en sistemas Red Hat, Fedora y SUSE:

  • rpm (bajo nivel): rpm -i/-U paquete.rpm (instalar/actualizar), rpm -e paquete (eliminar), rpm -qa (listar).
  • yum/dnf (Red Hat/Fedora): yum/dnf install paquete, yum/dnf update/upgrade paquete, yum/dnf remove paquete. Nota: yum update también actualiza la lista de paquetes disponibles desde los repositorios.
  • zypper (SUSE): zypper in paquete, zypper update, zypper rm paquete.
  • Repositorios en /etc/yum.repos.d/. Gestionamos con yum-config-manager, dnf config-manager, o zypper addrepo.

102.6 Linux como sistema virtualizado

Uso de Linux en entornos virtualizados:

  • Virtualización: Se pueden usar hipervisores como Xen (Tipo 1), KVM (Tipo 2, hospedado en el núcleo Linux), VirtualBox (Tipo 2). Tipos de máquinas: totalmente virtualizadas, paravirtualizadas, híbridas.
  • Plantillas: Nos ayudan a crear máquinas virtuales rápido. Revisamos que el D-Bus Machine ID sea único.
  • Nube: Hay IaaS (AWS, Azure). Gestionamos instancias, almacenamiento y redes. Accedemos con OpenSSH y configuramos con cloud-init.

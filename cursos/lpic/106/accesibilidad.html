106.3 Accesibilidad

Aprende a configurar opciones de accesibilidad en Linux.

La accesibilidad en Linux permite ajustar el sistema para usuarios con necesidades especiales.

Configuración de accesibilidad

Los entornos de escritorio tienen ajustes de accesibilidad:

  • Gnome: Acceso universal (menú en esquina superior).
  • KDE: Configuración del sistema → Accesibilidad.
  • Xfce: Ajustes gráficos → Accesibilidad (menos funciones).

Ejemplo: Sustituir alertas sonoras por visuales (campana visual en KDE).

Asistencia de teclado y mouse

Ajustes para movilidad reducida:

  • Teclas pegajosas: Permite atajos de teclas uno a uno (ej. Ctrl, luego C).
  • Teclas lentas: Requiere mantener tecla pulsada para aceptarla.
  • Teclas de rebote: Evita pulsaciones no intencionales con un retardo.
  • Teclas de conmutación: En KDE, pulsar 2 veces Alt, Ctrl o Shift las activa (como Bloq Mayús).
  • Teclas de repetición: Ajusta repetición automática al mantener tecla pulsada.
  • Teclas de ratón: Controla el puntero con teclado numérico (ej. 5 = clic izquierdo).
  • Gestos: Activa funciones (ej. Shift 5 veces para teclas pegajosas; mantener Shift 8 segundos para teclas lentas).
  • Teclado en pantalla: Usa ratón para escribir (Gnome: nativo; KDE: instalar onboard).

Nota: Estas opciones están en AccessX (extensión de X11), ajustables con xkbset.

Deficiencias visuales

Opciones para visión reducida:

  • Temas de escritorio de alto contraste/impresión grande: Colores nítidos y texto más grande.
  • Lupa de pantalla: Zoom para ampliar pantalla (Gnome: Zoom; KDE: KMagnifier; Xfce: Alt + rueda).
  • Lector de pantalla: Orca lee texto en voz alta (instalado por defecto).
  • Pantalla braille: Orca soporta dispositivos braille para lectura táctil.

Reconocimiento de voz: No está cubierto en el texto, pero herramientas externas pueden integrarse.

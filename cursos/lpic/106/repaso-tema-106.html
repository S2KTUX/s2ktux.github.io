106.1 Instalar y configurar X11

  • X11: Base gráfica en Linux (eventos de entrada, copiar/pegar, dibujar elementos).
  • Cliente/Servidor: Cliente (aplicación); Servidor (pantalla, teclado, ratón).
  • Configuración:
    • /etc/X11/xorg.conf: Archivo principal (opcional hoy), organizado en secciones que comienzan con Section "NombreSección" y terminan con EndSection, definiendo monitores, teclados, etc.
    • /etc/X11/xorg.conf.d/: Configuraciones adicionales (ej. teclado).
    • Generar: sudo Xorg -configure.
  • Conceptos clave:
    • ~/.xsession-errors: Log de errores.
    • xhost: Controla acceso (ej. xhost + permite conexiones locales).
    • xauth: Autenticación remota.
    • DISPLAY: Define la pantalla activa (ej. :0 para local, localhost:10.0 para remoto). Se configura automáticamente en SSH con X11 forwarding. Cada servidor X se identifica con un nombre de pantalla como :0 o :1.
    • X: Inicia/configura servidor.

106.2 Escritorios gráficos

  • Entornos:
    • Gnome: Gnome Shell (pantalla completa), usado en Fedora, Ubuntu.
    • KDE: KDE Plasma, usa Qt, en Kubuntu, openSUSE.
    • Xfce: Ligero, modular, para equipos con pocos recursos.
  • Componentes: Gestor de ventanas (controla ventanas), widgets (botones, campos), apps básicas (terminal, navegador).
  • Acceso remoto:
    • XDMCP: Pantallas remotas, inseguro, alto ancho de banda.
    • VNC: Usa RFB, puertos 5900+, sin cifrado (usar SSH).
    • RDP: Acceso a Windows, puerto 3389.
    • Spice: Para sistemas virtualizados, integra dispositivos locales como USB (memorias, cámaras) y permite acceso a interfaces gráficas remotas. Diseñado para la salida de video de máquinas virtuales, ofrece una experiencia gráfica fluida.

106.3 Accesibilidad

  • Ajustes: Gnome (Acceso universal), KDE (Configuración → Accesibilidad), Xfce (menos opciones). Gestores de pantalla como GDM o SDDM inician el entorno de escritorio y gestionan el login, pudiendo activar funciones de accesibilidad como lectores de pantalla.
  • Teclado/Mouse:
    • Teclas pegajosas: Atajos uno a uno (ej. Ctrl, luego C).
    • Teclas lentas: Mantener tecla para aceptar.
    • Teclas de rebote: Retardo para evitar pulsaciones accidentales.
    • Teclas de ratón: Mover puntero con teclado numérico.
    • Teclado en pantalla: Escribir con ratón (Gnome: nativo; KDE: onboard).
    • Gestos: Activar funciones (ej. Shift 5 veces).
  • Visión:
    • Temas de alto contraste/impresión grande: Mejora visibilidad.
    • Lupa de pantalla: Zoom (Gnome: Zoom; KDE: KMagnifier).
    • Lector de pantalla: Orca convierte texto mostrado en voz o Braille, permitiendo interacción a usuarios ciegos o con visión reducida.
    • Pantalla braille: Orca soporta dispositivos que representan texto con puntos elevados táctiles para lectura por usuarios con discapacidad visual.

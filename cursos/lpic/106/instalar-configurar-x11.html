106.1 Instalar y configurar X11

Aprende cómo funciona y se configura el sistema X Window en Linux.

X11 es la base de las interfaces gráficas en Linux. Configurarlo es esencial para entornos gráficos.

Arquitectura de X Window

X11 divide el trabajo entre:

  • Cliente: Aplicaciones (ej. navegador, terminal) que deciden qué dibujar.
  • Servidor: Muestra el contenido en pantalla y gestiona entrada (teclado, ratón).

Estructura básica

X11 es modular y soporta redes:

  • Clientes remotos pueden usar un servidor X.
  • Extensiones (ej. libX11) añaden funciones sin cambiar el núcleo.
  • Administradores: Inician X (ej. GDM, LightDM).
  • Nombres de pantalla: Formato hostname:displaynumber.screennumber (ej. :0).

Configuración de X

Configura X11 con:

  • /etc/X11/xorg.conf: Archivo principal (opcional en sistemas modernos).
  • /etc/X11/xorg.conf.d/: Directorio para configuraciones adicionales (ej. teclados).
  • Genera con: 
    sudo Xorg -configure
    y mueve a /etc/X11/xorg.conf.
  • Revisa con: 
    xdpyinfo
    (muestra versión, pantalla).

Conceptos clave

Domina estos términos para el examen:

  • /etc/X11/xorg.conf: Archivo de configuración principal de X11.
  • /etc/X11/xorg.conf.d/: Directorio para ajustes específicos.
  • ~/.xsession-errors: Log de errores de la sesión X.
  • xhost: Controla acceso al servidor (ej. xhost +).
  • xauth: Autenticación para conexiones remotas.
  • DISPLAY: Variable de pantalla (ej. echo $DISPLAY → :0).
  • X: Comando para iniciar o configurar X (ej. X -configure).

Tema 105: Shells y Scripts (LPIC-1)

Domina el entorno de shell y aprende a escribir scripts en Linux para automatizar tareas.

← Volver a Cursos

Descripción

El Tema 105 de LPIC-1 te enseña cómo personalizar el entorno de shell y escribir scripts básicos en Linux. Aprenderás a configurar tu shell para mayor eficiencia y a automatizar tareas con scripts sencillos, habilidades esenciales para un administrador de sistemas.

Conceptos que aprenderás

En este tema, explorarás:

  • Configuración del entorno de shell con archivos como .bashrc y .bash_profile.
  • Definición de variables de entorno, alias y funciones para personalizar el shell.
  • Estructura y sintaxis de scripts básicos en bash.
  • Uso de estructuras de control (como if, for, while) y variables en scripts.
  • Permisos y ejecución de scripts en Linux.

Acceder a una solicitud de shell y emitir comandos con la sintaxis correcta

El acceso a una shell puede realizarse desde consola local usando Ctrl + Alt + F1–F6 o de forma remota mediante SSH.

La sintaxis general de los comandos en Linux es:

comando [opciones] [argumentos]

Ejemplo:

ls -l /ruta

Utilizar la redirección de entrada y salida (>, >>, |, 2>, etc.)

En Linux todo es un archivo y cada proceso utiliza descriptores estándar:

  • 0 → STDIN
  • 1 → STDOUT
  • 2 → STDERR
cat /etc/passwd > archivo.txt
ls /ruta 2> error.txt
ls /noexiste /etc > salida.txt 2>&1
ls -l | grep ".conf"

Usar grep y expresiones regulares para analizar texto

grep permite buscar patrones dentro de archivos o salidas de comandos.

grep root /etc/passwd

Expresiones regulares básicas:

^root
bash$
r..t
[0-9]

Acceder a sistemas remotos mediante SSH

SSH permite conectarse de forma segura a sistemas remotos.

ssh usuario@servidor
ssh -p 2222 usuario@servidor

Iniciar sesión y cambiar de usuario en destinos multiusuario

Para cambiar de usuario se utilizan los comandos su y sudo.

su usuario
su - usuario
sudo -i

Archivar, comprimir, desempaquetar y descomprimir archivos utilizando tar, gzip y bzip2

tar se utiliza para empaquetar archivos, mientras que gzip y bzip2 comprimen.

tar -cvf archivo.tar archivo
tar -czvf archivo.tar.gz carpeta/
tar -xvf archivo.tar

Crear y editar archivos de texto

Los archivos pueden crearse y editarse con distintas herramientas.

touch archivo.txt
nano archivo.txt
vi archivo.txt

Crear, eliminar, copiar y mover archivos y directorios

Linux dispone de comandos básicos para la gestión de archivos.

mkdir carpeta
rm archivo.txt
cp origen destino
mv archivo nuevo

Crear enlaces físicos y simbólicos

Los enlaces permiten referenciar archivos y directorios.

ln archivo enlace_fisico
ln -s archivo enlace_simbolico

Enumerar, configurar y cambiar permisos estándar ugo/rwx

Los permisos controlan el acceso a archivos y directorios.

ls -l
chmod 755 archivo
chown usuario:grupo archivo

Localizar, leer y utilizar la documentación del sistema, incluyendo man, info y archivos en /usr/share/doc

Linux dispone de documentación integrada para casi todos los comandos.

man ls
info ln
apropos passwd
whatis mv

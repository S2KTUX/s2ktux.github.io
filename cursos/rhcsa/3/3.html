Ejecutar código condicionalmente

Los scripts de shell permiten ejecutar comandos de forma automática. El control condicional se realiza principalmente con if, test y [].

Estructura básica de un script:

#!/bin/bash
# Comentarios
echo "Hola mundo"

Ejemplo de uso de if con test:

#!/bin/bash

if test -f /etc/passwd; then
    echo "El archivo existe"
else
    echo "El archivo no existe"
fi

Forma equivalente usando [] (la más común):

if [ -f /etc/passwd ]; then
    echo "Existe"
fi

Operadores comunes:

  • -f → Archivo existe y es regular
  • -d → Directorio existe
  • -eq, -ne, -gt, -lt → Comparaciones numéricas
  • =, != → Comparaciones de cadenas

⚠️ En el examen RHCSA es habitual comprobar la existencia de archivos, directorios o el valor de variables.

Uso de bucles para procesar archivos y entradas

Los bucles permiten repetir acciones automáticamente. El más usado es for.

Ejemplo: recorrer una lista de archivos:

#!/bin/bash

for archivo in /etc/*.conf; do
    echo "Procesando $archivo"
done

Ejemplo: procesar argumentos de línea de comandos:

for usuario in "$@"; do
    echo "Usuario: $usuario"
done

Ejemplo con while leyendo entrada estándar:

while read linea; do
    echo "Línea: $linea"
done < archivo.txt

Estos bucles son muy útiles para tareas típicas de administración: crear usuarios, revisar permisos o procesar logs.

Procesar entradas de scripts

Los scripts pueden recibir argumentos desde la línea de comandos:

  • $0 → Nombre del script
  • $1, $2, $3… → Argumentos individuales
  • $# → Número total de argumentos
  • $@ → Todos los argumentos

Ejemplo básico:

#!/bin/bash

echo "Primer argumento: $1"
echo "Segundo argumento: $2"
echo "Total: $#"

Ejemplo validando argumentos:

if [ $# -ne 1 ]; then
    echo "Uso: $0 archivo"
    exit 1
fi

Este tipo de validación es muy común en el examen RHCSA.

Procesar la salida de comandos dentro de un script

La salida de comandos puede almacenarse en variables usando $(comando) o comillas invertidas.

Ejemplo:

#!/bin/bash

USUARIOS=$(wc -l /etc/passwd | awk '{print $1}')
echo "Usuarios en el sistema: $USUARIOS"

Otro ejemplo comprobando servicios:

ESTADO=$(systemctl is-active sshd)

if [ "$ESTADO" = "active" ]; then
    echo "SSHD está activo"
else
    echo "SSHD no está activo"
fi

Esta técnica permite tomar decisiones basadas en el estado real del sistema, algo clave para automatización.

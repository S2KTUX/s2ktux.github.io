pwd · cd · ls -laRuta actual, cambiar de directorio, listar (oculto/permisos con -la).
cd /Ve a la raíz: verás un árbol Linux real (/etc, /var, /usr, /home...).
tree · cat · head · tailÁrbol; ver contenido; primeras/últimas líneas (-n N).
mkdir · rmdir · touchCrear directorio, borrar vacío, crear fichero.
rm -r · cp · mvBorrar (recursivo), copiar, mover/renombrar.
echo txt > f · >>Escribir en fichero (> sobrescribe, >> añade).
find . -name x · grepBuscar ficheros por nombre; buscar texto dentro.
cmd | grep · wc · headTuberías reales: encadena la salida de un comando a otro.
./script.shEjecuta un script (con permiso +x): corre sus líneas de verdad.
chmod 755 / u+xCambia permisos; se refleja al instante en ls -l.
chown user:grp fCambia propietario y grupo.
chgrp grp f · umaskCambia el grupo / máscara por defecto.
whoami · id · groupsUsuario actual, UID/GID, grupos.
useradd nombreCrea usuario: aparece en cat /etc/passwd y se le crea /home.
userdel · usermod -aGBorra usuario / añade a un grupo.
groupadd · passwdCrea grupo / cambia contraseña.
su - nombre · exitCambia de usuario (prompt pasa a nombre@s2ktux); exit vuelve.
cat /etc/passwdLista de usuarios del sistema (real y actualizada).
ps -ef · ps aux · topTodos los procesos con formato Linux real. ps solo = tu terminal.
kill · pgrepTerminar un proceso por PID / buscar por nombre.
systemctl start/enable sGestiona servicios; el proceso aparece luego en ps.
systemctl status sEstado detallado de un servicio.
journalctl -u sLogs de un servicio.
uptime · free -hTiempo encendido / memoria.
dnf install pInstala un paquete (si existe en el repo). Registra lo instalado.
dnf search · info pBusca / muestra información de un paquete.
dnf list installed/availableLista lo instalado o lo disponible en el repo.
dnf remove · updateDesinstala / actualiza paquetes.
rpm -qa · rpm -q pConsulta los paquetes rpm instalados.
apt install pGestor de Debian/Ubuntu (equivalente).
lsblk · fdisk -l · blkidDiscos y particiones; tabla; UUIDs. Hay un disco vacío /dev/sdb.
parted /dev/sdb mklabel gptCrea la tabla de particiones en el disco vacío.
parted /dev/sdb mkpart ...mkpart primary xfs 0% 100% → crea /dev/sdb1.
mkfs.xfs /dev/sdb1Formatea la partición (también mkfs.ext4).
mount /dev/sdb1 /mnt/dataMonta; aparece en df -h y lsblk. umount para desmontar.
df -h · du -sh · cat /etc/fstabUso de disco / tamaño / montajes persistentes.
pvs · vgs · lvsLVM: volúmenes físicos, grupos y lógicos.
ip a · ip routeInterfaces y direcciones / tabla de rutas.
ip link set eth0 down/upTumba o levanta la interfaz. ¡Con eth0 down el ping externo falla!
ping hostTrazas reales una a una; localhost siempre responde.
ss -tuln · nmcliPuertos en escucha / gestión de conexiones.
firewall-cmd · hostnamectlFirewall (firewalld) / nombre del host.
docker ps -a · imagesContenedores / imágenes (podman funciona igual).
docker pull · run -dDescarga una imagen / crea y arranca un contenedor.
docker start/stop/restartControla el ciclo de vida de un contenedor.
docker logs · inspect · execLogs, detalles y shell dentro de un contenedor.
docker build -t n . · rmiConstruye una imagen / borra una imagen.
podman ...Mismo interfaz que docker, sin daemon (rootless).
⏳ El módulo interactivo de Kubernetes llegará muy pronto al sandbox. De momento, esta es la referencia de comandos.
kubectl get pods/nodes/svcLista recursos del clúster (add -A o -n ns).
kubectl run · createCrea un pod / recurso rápidamente.
kubectl apply -f fAplica un manifiesto YAML.
kubectl expose · scaleExpone un servicio / escala réplicas.
kubectl describe · logs · execDetalle, logs y shell de un pod.
kubectl delete ...Elimina recursos.

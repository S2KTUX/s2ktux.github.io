💡 Escribe man <comando> en la terminal para ver todas sus opciones. Aquí solo va el comando y para qué sirve.
pwd · cd · lsRuta actual, cambiar de directorio, listar.
tree · cat · head · tailÁrbol, ver contenido, principio/final.
mkdir · rmdir · touchCrear directorio / borrar vacío / crear fichero.
rm · cp · mv · lnBorrar, copiar, mover, enlazar.
find · grep · wcBuscar ficheros / texto / contar.
stat · file · whichMetadatos / tipo / ruta del ejecutable.
sort · uniq · nlOrdenar / quitar duplicados / numerar.
echo texto > f · >>Escribir en fichero (sobrescribe / añade).
cmd1 | cmd2Tuberías: encadena la salida de un comando a otro.
cmd1 && cmd2 · || · ;Encadena por éxito / fallo / siempre.
vi FICHERO · nano FICHEROEditores (vi modal; nano con ^O/^X).
./script.shEjecuta un script con permiso +x.
ls -lLee permisos: rwx para usuario/grupo/otros. Ej: -rwxr-x--- = 750.
chmod 640 F · chmod u+x FOctal (r=4 w=2 x=1) o simbólico (u/g/o + +/- + rwx). Ej: 640 = rw-r-----.
chmod -R g+w DIR-R aplica recursivamente a todo el árbol.
chown usuario:grupo FCambia propietario y grupo (chgrp solo el grupo).
umaskMáscara por defecto (022 → ficheros 644, directorios 755).
chmod u+s · g+s · +tEspeciales: SUID / SGID (herencia de grupo en dirs) / sticky bit.
getfacl FMuestra las ACL de un fichero (permisos más allá de ugo).
setfacl -m u:ana:rwx FDa permisos a un usuario concreto (-m modifica, -x quita, -b borra todas).
whoami · id · groupsQuién soy, UID/GID, grupos.
useradd · userdel · usermodCrear / borrar / modificar usuarios.
groupadd · passwd · chageCrear grupo / contraseña / caducidad.
su · exit · getentCambiar de usuario / volver / consultar BD.
ps · top · pgrep · killVer procesos / en vivo / buscar / matar.
systemctlServicios: status/start/stop/enable/restart.
journalctlLogs de servicios y del sistema.
crontab · uptime · freeTareas programadas / carga / memoria.
dnf search TEXTO · dnf info PAQBusca en los repos y muestra detalles de un paquete.
dnf install PAQ · -yInstala (pide confirmación; -y la salta). Ej: dnf install httpd.
dnf remove · dnf updateElimina o actualiza todo el sistema.
dnf list installed · availableLista paquetes instalados o disponibles.
dnf provides /ruta · repolistQué paquete da un fichero · repos activos.
dnf config-manager --add-repo URLAñade un repositorio en /etc/yum.repos.d/.
dnf group list · group installGrupos de paquetes (p.ej. "Development Tools").
rpm -qa · rpm -q PAQLista todo lo instalado · consulta un paquete.
rpm -ql PAQ · rpm -qf /rutaFicheros de un paquete · a qué paquete pertenece un fichero.
rpm -qi PAQ · rpm -V PAQInformación detallada · verifica integridad.
rpm -ivh FICHERO.rpmInstala un .rpm local (para RHCSA; dnf resuelve dependencias).
yumObsoleto en RHEL 9: usa dnf (apt/dpkg no existen en Red Hat).
lsblk · fdisk · blkidDiscos, particiones y UUIDs. Hay un /dev/sdb vacío.
partedParticionado (también discos grandes / GPT).
mkfs.xfs · mkfs.ext4Formatea una partición.
mount · umount · df · duMonta / desmonta / uso de disco.
pvcreate · vgcreate · vgextendLVM: inicia PV, crea/amplía el grupo (VG). Compruébalo con pvs / vgs.
lvcreate -n x -L 5G vgCrea un volumen lógico (· -l 100%FREE usa todo el espacio libre).
lvextend -r -L +2G /dev/vg/xAmplía el LV y, con -r, también el FS (o xfs_growfs / resize2fs).
pvs vgs lvs · lvremoveVer PV/VG/LV · eliminar un volumen.
mkswap · swapon --showCrear y activar/ver el swap.
ip a · ip routeDirecciones IP e interfaces · tabla de rutas (puerta de enlace).
ip link set eth0 up|downActiva o desactiva el enlace (equivale a ifup/ifdown).
nmcli general · dev statusEstado de red y de los dispositivos (connected/disconnected).
nmcli con show [eth0]Lista conexiones · detalle completo de una (IP, gateway, DNS).
nmcli con mod eth0 ipv4.method manualFija IP estática (método manual).
... ipv4.addresses 10.0.0.5/24Dirección y máscara. Añade ipv4.gateway y ipv4.dns.
nmcli con up eth0Aplica los cambios (obligatorio tras con mod).
nmtuiConfiguración de red por menús (editar/activar/hostname).
ping 8.8.8.8 · ping HOSTPrueba conectividad. Falla si falta enlace, gateway o DNS.
ss -tlnpPuertos TCP en escucha y el proceso que los abre.
firewall-cmd --list-allFirewalld: servicios/puertos. --add-* --permanent y --reload.
hostnamectl set-hostname NOMBREFija el nombre del sistema de forma persistente.
getenforce · sestatusModo actual / estado de SELinux.
setenforce 0|1Permissive / Enforcing.
getsebool · setseboolVer / cambiar booleanos.
semanage · restorecon · chconPuertos, contextos y su restauración.
ls -Z · ls -lZContexto SELinux de los ficheros (· -lZ en formato largo).
restorecon -Rv RUTARestaura el contexto correcto (p.ej. tras tocar /etc/shadow en rescate).
touch /.autorelabelFuerza el reetiquetado de todo el FS en el próximo arranque.
docker run · ps · logsArranca contenedores, lista, logs.
start · stop · restart · rmCiclo de vida de un contenedor.
exec · attach · inspect · topEntrar, detalles y procesos.
images · pull · searchImágenes locales, descargar, buscar.
build · commit · historyConstruir desde Dockerfile / capas.
tag · push · save · loadEtiquetar, publicar, exportar/importar.
volume · networkVolúmenes y redes (bridge/host/none/overlay/macvlan).
stats · system df · pruneConsumo, uso de disco, limpieza.
run -m --cpus --restartLímites de recursos y política de reinicio.
podmanIgual que docker, sin daemon (rootless).
docker compose up -d · downArranca / para toda la app multi-contenedor.
compose ps · logs · buildEstado / logs / construir servicios.
docker-composeForma antigua, equivalente.
Ejemplo docker-compose.yml:
services:
  web:
    image: nginx
    ports: ["8080:80"]
  db:
    image: mariadb:11
    environment: ["MARIADB_ROOT_PASSWORD=secreto"]
    volumes: ["db_data:/var/lib/mysql"]
man · helpManual de un comando / índice rápido.
uname · hostnamectlKernel e información del host.
date · timedatectlFecha, hora y zona.
uptime · free · env · historyCarga, memoria, variables, historial (· !! repite el último, !texto el último que empieza por, Ctrl+R busca).
tuned-adm · tar · neofetchPerfiles, comprimir, resumen del sistema.
clear (Ctrl+L) · resetLimpia pantalla / reinicia el entorno.
